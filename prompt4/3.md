# 3. 챗 요약 및 인사이트 추출 AI

> API: `POST /chat/sessions/{sessionId}/summarize`  
> 목적: 챗봇 대화를 활동 후보 생성에 사용할 수 있는 요약 정보로 정리한다.

## 3.1 System Prompt

```txt
너는 대학생의 진로 상담 대화를 요약하는 분석가입니다.
아래 채팅 내용을 바탕으로 사용자의 목표, 걱정, 선호 활동, 제약 조건을 추출합니다.

규칙:
- 사용자가 말하지 않은 내용은 지어내지 않습니다.
- 문장은 짧고 구체적으로 작성합니다.
- 활동 후보 생성 AI가 바로 사용할 수 있게 정리합니다.
- goals, concerns, preferredActivities, constraints는 각각 string 배열로 작성합니다.
- 정보가 부족하면 빈 배열로 둡니다.
```

## 3.2 User Message

```txt
[프로필]
{profile}

[성향 설문 결과]
{surveyResult}

[채팅 메시지 목록]
{messages}

위 대화를 요약하고, 활동 후보 생성에 필요한 인사이트를 추출해줘.
```

## 3.3 호출 설정

```json
{
  "model": "gemini",
  "temperature": 0.3,
  "output_config": {
    "format": {
      "type": "json_schema",
      "schema": {
        "type": "object",
        "properties": {
          "summary": { "type": "string" },
          "insights": {
            "type": "object",
            "properties": {
              "goals": {
                "type": "array",
                "items": { "type": "string" }
              },
              "concerns": {
                "type": "array",
                "items": { "type": "string" }
              },
              "preferredActivities": {
                "type": "array",
                "items": { "type": "string" }
              },
              "constraints": {
                "type": "array",
                "items": { "type": "string" }
              }
            },
            "required": [
              "goals",
              "concerns",
              "preferredActivities",
              "constraints"
            ],
            "additionalProperties": false
          }
        },
        "required": ["summary", "insights"],
        "additionalProperties": false
      }
    }
  }
}
```

## 3.4 기대 출력 예시

```json
{
  "summary": "사용자는 졸업 전 해외 경험과 실무 경험을 원하며, 낮은 학점과 준비 부족을 걱정하고 있습니다.",
  "insights": {
    "goals": ["졸업 전 해외 경험 만들기", "실무 경험 확보"],
    "concerns": ["학점이 낮아 지원 가능성이 낮을까 걱정함", "준비 부족감"],
    "preferredActivities": ["해외 인턴", "실무 프로젝트", "어학 준비"],
    "constraints": ["학기 중에는 학점 관리 필요", "방학에 집중 활동 가능"]
  }
}
```
