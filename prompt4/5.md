# 5. 학기별 계획 생성 AI

> API: `POST /plans/generate`  
> 목적: 선택된 활동 후보를 바탕으로 남은 학기별 로드맵을 생성한다.

## 5.1 System Prompt

```txt
너는 대학생의 남은 학기와 방학을 기준으로 수업 밖 활동 시간표를 설계하는 AI입니다.
선택된 활동 후보를 바탕으로 3학년 1학기부터 4학년 2학기까지의 학기별 계획을 만듭니다.

규칙:
- 한 시기에 너무 많은 활동을 몰아넣지 않습니다.
- career_required 활동을 우선 배치합니다.
- life_required 활동은 장기적으로 유지되도록 배치합니다.
- 방학에는 인턴, 프로젝트, 포트폴리오처럼 시간이 필요한 활동을 배치합니다.
- 학기 중에는 수업 부담을 고려해 무리하지 않게 배치합니다.
- 사용자의 제약 조건을 반영합니다.
- periodKey는 3-1, summer, 3-2, winter, 4-1, final_summer, 4-2 중 하나입니다.
- 각 plan item은 실행 가능한 제목과 설명으로 작성합니다.
```

## 5.2 User Message

```txt
[프로필]
{profile}

[성향 설문 결과]
{surveyResult}

[챗봇 요약]
{chatSummary}

[선택된 활동 후보]
{selectedCandidates}

[계획 타입]
{planType}

선택된 활동 후보를 바탕으로 남은 학기별 계획을 생성해줘.
```

## 5.3 호출 설정

```json
{
  "model": "gemini",
  "temperature": 0.4,
  "output_config": {
    "format": {
      "type": "json_schema",
      "schema": {
        "type": "object",
        "properties": {
          "plan": {
            "type": "object",
            "properties": {
              "title": { "type": "string" },
              "type": { "type": "string" },
              "status": { "type": "string" }
            },
            "required": ["title", "type", "status"],
            "additionalProperties": false
          },
          "items": {
            "type": "array",
            "items": {
              "type": "object",
              "properties": {
                "periodKey": { "type": "string" },
                "periodLabel": { "type": "string" },
                "title": { "type": "string" },
                "description": { "type": "string" },
                "category": { "type": "string" },
                "sourceActivityId": { "type": "string" },
                "orderIndex": { "type": "number" }
              },
              "required": [
                "periodKey",
                "periodLabel",
                "title",
                "description",
                "category",
                "sourceActivityId",
                "orderIndex"
              ],
              "additionalProperties": false
            }
          }
        },
        "required": ["plan", "items"],
        "additionalProperties": false
      }
    }
  }
}
```
