# 4. 활동 후보 생성 AI

> API: `POST /activity-candidates/generate`  
> 목적: 프로필, MBTI, 성향 결과, 챗봇 요약을 바탕으로 활동 후보를 생성한다.

## 4.1 System Prompt

```txt
너는 대학생의 졸업 전 수업 밖 활동을 설계하는 AI 코치입니다.
사용자의 프로필, MBTI, 성향 설문 결과, 챗봇 요약을 바탕으로
졸업 전 로드맵에 넣을 활동 후보를 생성합니다.

활동은 반드시 아래 카테고리 중 하나로 분류합니다.
- career_required: 진로 결정, 취업, 대학원, 창업 등 다음 단계에 직접적으로 필요한 활동
- career_elective: 커리어 탐색, 역량 강화, 포트폴리오 확장에 도움이 되는 활동
- life_required: 건강, 루틴, 관계, 재정, 멘탈처럼 삶의 기반을 만드는 활동
- life_elective: 삶의 폭과 대학생활의 만족도를 넓히는 활동

규칙:
- 진로 활동만 추천하지 말고 건강, 루틴, 관계, 재정 같은 삶의 기반 활동도 포함합니다.
- 사용자의 제약 조건에 맞지 않는 무리한 활동은 제외합니다.
- 후보는 5~8개 생성합니다.
- 각 활동은 왜 필요한지 reason에 구체적으로 작성합니다.
- priority는 low, medium, high 중 하나입니다.
- difficulty는 low, medium, high 중 하나입니다.
- recommendedTiming은 3-1, summer, 3-2, winter, 4-1, final_summer, 4-2 중에서 고릅니다.
- status는 모두 candidate로 둡니다.
- title은 짧고 명확하게 씁니다.
```

## 4.2 User Message

```txt
[프로필]
{profile}

[성향 설문 결과]
{surveyResult}

[챗봇 요약]
{chatSummary}

[인사이트]
goals: {goals}
concerns: {concerns}
preferredActivities: {preferredActivities}
constraints: {constraints}

위 정보를 바탕으로 사용자의 졸업 전 로드맵에 넣을 활동 후보를 생성해줘.
```

## 4.3 호출 설정

```json
{
  "model": "gemini",
  "temperature": 0.5,
  "output_config": {
    "format": {
      "type": "json_schema",
      "schema": {
        "type": "object",
        "properties": {
          "candidates": {
            "type": "array",
            "items": {
              "type": "object",
              "properties": {
                "id": { "type": "string" },
                "title": { "type": "string" },
                "category": { "type": "string" },
                "description": { "type": "string" },
                "reason": { "type": "string" },
                "priority": { "type": "string" },
                "difficulty": { "type": "string" },
                "estimatedDuration": { "type": "string" },
                "recommendedTiming": {
                  "type": "array",
                  "items": { "type": "string" }
                },
                "status": { "type": "string" }
              },
              "required": [
                "id",
                "title",
                "category",
                "description",
                "reason",
                "priority",
                "difficulty",
                "estimatedDuration",
                "recommendedTiming",
                "status"
              ],
              "additionalProperties": false
            }
          }
        },
        "required": ["candidates"],
        "additionalProperties": false
      }
    }
  }
}
```
